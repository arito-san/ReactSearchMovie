## ReactSearchMovie
This is a web application that provides movie and poster data from a public api!

## Technology 
Here are the technologies used in this project:
* React 18.2.0
* JavaScript
* LottieFiles
* axios

## Services Used
* Github
* The Open Movie Database (OMDb)
